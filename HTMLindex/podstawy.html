Sekcja <head></head>

Elementy <head>


  • <meta charset="UTF-8">
  • <title></title>
  • <meta name="description" content="...">
  • <meta name="keywords" content="...">
  • <meta name="author" content="...">
  • <link rel="stylesheet" href="...">
  • <link rel="icon" href="...">

Zwyczajowo pierwszym elementem, który wstawia się do tej sekcji, jest deklaracja zestawu znaków, który będzie używany na stronie internetowej, aby tekst wyświetlał się odpowiednio.

Element ten to tytuł poszczególnej strony internetowej. Nie jest on jednak wyświetlany w treści strony, dlatego też zamieścimy go w sekcji <head>.

Kolejnym bardzo istotnym dla Google elementem meta jest description, czyli po polsku opis. W tym przypadku opis strony.

Meta name keywords, czyli meta tag ze słowami kluczowymi.

Meta name author zawiera informacje o autorze strony.

Element używany jest w celu importu zewnętrznych zewnętrznych plików na stronę internetową.
- Import zewnętrznego pliku CSS.

- Dołączenie Favicona.
